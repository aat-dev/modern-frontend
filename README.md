modern frontend
===============

Simple development starting point for modern frontend web apps based on javascript.

Tools used : 

    - AngularJS 1.4.x
    - Bower
    - Grunt
    - Bootstrap 3.x 
    - SASS
    - jQuery 2.x
    
    
Testing : 

    - Karma
    - Jasmine
    - Protractor
